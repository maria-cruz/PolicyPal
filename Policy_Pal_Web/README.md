## Getting Started

1.  **Installation**

    Clone and run npm install to install the dependencies.

    ```bash
    npm install
    ```



1.  **Start locally**

    Start the application in watch mode.

    ```bash
    npm start
    ```

1.  **Open the source code and start editing!**

    Your site is now running at `http://localhost:3000`!
